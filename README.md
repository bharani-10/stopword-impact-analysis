## 📌 Project Overview
This project analyzes the impact of stopword removal on text classification performance in Natural Language Processing (NLP).

The system compares how machine learning models behave when:
- Stopwords are retained
- Stopwords are removed

It demonstrates how preprocessing affects prediction accuracy, token representation, and model efficiency.

---

## 🎯 Objectives
- To implement text preprocessing techniques
- To compare text classification with and without stopwords
- To evaluate model performance differences
- To visualize token reduction using graphs

---

## 🧠 Models Used
- Naive Bayes (for probabilistic classification)
- Logistic Regression (for linear classification)

## 🔄 Project Workflow

1. Data Collection  
   Dataset containing spam and ham messages is used.

2. Data Preprocessing  
   - Lowercasing  
   - Removing punctuation  
   - Tokenization  

3. Stopword Handling  
   - Version 1: With stopwords  
   - Version 2: Without stopwords  

4. Feature Extraction  
   - Text converted into numerical form using vectorization

5. Model Training  
   - Naive Bayes and Logistic Regression models trained

6. Prediction  
   - Classifies text as Spam or Ham

7. Visualization  
   - Graph comparing token count before and after stopword removal
