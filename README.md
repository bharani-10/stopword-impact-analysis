# Stopword Removal Impact Analysis

This project analyzes the impact of stopword removal in Natural Language Processing (NLP). It compares text classification performance using datasets with and without stopwords.

Models used:
- Naive Bayes
- Logistic Regression

Features:
- Text preprocessing
- Stopword removal
- Bag of Words & TF-IDF
- Model training and evaluation
- Streamlit web application

The project demonstrates how preprocessing techniques influence model performance in spam detection tasks.
